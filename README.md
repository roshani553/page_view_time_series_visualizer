This project analyzes and visualizes the daily page views of the freeCodeCamp forum from **2016 to 2019** using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**.

#Features

* Data cleaning (removes outliers)
* Line plot of daily page views
* Bar plot showing monthly averages per year
* Box plots for year-wise and month-wise trends

# Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

# How to Run

1. Download the dataset file `fcc-forum-pageviews.csv`
2. Run the Python file:

   ```bash
   python page_view_time_series_visualizer.py
   ```
3. View the generated graphs
